# Dataset Information

## 📊 Dataset Name
Sonar Dataset

---

## 📌 Description
This dataset contains sonar signal measurements used to classify objects as either rocks or mines. The dataset consists of multiple numerical features extracted from sonar returns and is widely used for Machine Learning classification and Neural Network applications.

---

## 📂 File Included
- `sonardataset.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| Feature_0 to Feature_59 | Numerical sonar signal attributes representing energy values in different frequency bands |
| Target | Object classification label (R = Rock, M = Mine) |

---

## 🛠️ Usage
This dataset is used for:
- Neural Networks
- Binary Classification
- Deep Learning
- Machine Learning Model Training
- Feature Analysis
- Exploratory Data Analysis (EDA)
