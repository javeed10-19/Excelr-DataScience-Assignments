# Dataset Information

## 📊 Dataset Name
Fetal Health Dataset

---

## 📌 Description
This dataset contains fetal cardiotocography measurements used for analyzing fetal health conditions. The dataset includes various medical attributes related to fetal heart rate and movement patterns, which are useful for statistical analysis, Exploratory Data Analysis (EDA), and Machine Learning practice.

---

## 📂 File Included
- `fetal_health.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| LB | Baseline Fetal Heart Rate |
| AC | Accelerations |
| FM | Fetal Movements |
| UC | Uterine Contractions |
| DL | Light Decelerations |
| DS | Severe Decelerations |
| DP | Prolonged Decelerations |
| ASTV | Abnormal Short Term Variability |
| MSTV | Mean Short Term Variability |
| ALTV | Abnormal Long Term Variability |
| MLTV | Mean Long Term Variability |
| Width | Histogram Width |
| Tendency | Histogram Tendency |
| NSP | Fetal Health Status |
