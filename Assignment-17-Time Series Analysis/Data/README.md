# Dataset Information

## 📊 Dataset Name
Exchange Rate Dataset

---

## 📌 Description
This dataset contains historical exchange rate values recorded over time. It is commonly used for Time Series Analysis, trend identification, seasonality analysis, and forecasting future exchange rate movements.

---

## 📂 File Included
- `exchange_rate.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| date | Date and time of the exchange rate observation |
| Ex_rate | Exchange rate value recorded on the given date |

---

## 🛠️ Usage
This dataset is used for:
- Time Series Analysis
- Trend Analysis
- Forecasting
- Data Visualization
- Seasonal Pattern Detection
- Financial Data Analysis
