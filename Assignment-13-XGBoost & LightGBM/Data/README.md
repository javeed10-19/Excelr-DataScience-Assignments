# Dataset Information

## 📊 Dataset Name
Diabetes Dataset

---

## 📌 Description
This dataset contains medical diagnostic measurements used to predict whether a patient has diabetes. The dataset is widely used for Machine Learning classification and healthcare prediction analysis.

---

## 📂 File Included
- `diabetes.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| Pregnancies | Number of times pregnant |
| Glucose | Glucose concentration level |
| BloodPressure | Blood pressure measurement |
| SkinThickness | Thickness of skin fold |
| Insulin | Insulin level in blood |
| BMI | Body Mass Index |
| DiabetesPedigreeFunction | Diabetes hereditary function score |
| Age | Age of the patient |
| Outcome | Diabetes prediction result (0 = No Diabetes, 1 = Diabetes) |

---

## 🛠️ Usage
This dataset is used for:
- Logistic Regression
- Classification Problems
- Machine Learning Model Training
- Healthcare Prediction
- Exploratory Data Analysis (EDA)
