## 📊 Problem Statement
Bombay Hospitality Ltd. uses the operating cost model:

\[
W = 1000 + 5X
\]

where:
- \(W\) = Weekly Operating Cost
- \(X\) = Number of units produced

The restaurant owners claim that the actual operating costs are higher than the theoretical model.

A hypothesis test is conducted using:
- Sample Mean = Rs. 3050
- Population Mean based on model
- Standard Deviation
- Sample Size = 25
- Significance Level = 5%
