# Dataset Information

## 📊 Dataset Name
Amazon Reviews Dataset

---

## 📌 Description
This dataset contains customer reviews from Amazon products along with sentiment labels. The dataset is commonly used for Natural Language Processing (NLP), Sentiment Analysis, Text Classification, and Machine Learning applications.

---

## 📂 File Included
- `amazonreviews.tsv`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| label | Sentiment label of the review (`pos` = Positive, `neg` = Negative) |
| review | Customer review text |

---

## 🛠️ Usage
This dataset is used for:
- Natural Language Processing (NLP)
- Sentiment Analysis
- Text Classification
- Text Mining
- Machine Learning Model Training
- Feature Extraction from Text

---

Dataset provided as part of the ExcelR Data Science course assignments.
