# Dataset Information

## 📊 Dataset Name
Pharma Industry Drug Response Dataset

---

## 📌 Description
This dataset contains pharmaceutical and patient health-related measurements used to analyze drug response outcomes. The dataset includes medical parameters such as drug dosage, blood pressure, heart rate, liver toxicity, and blood glucose levels for Machine Learning and healthcare analytics practice.

---

## 📂 File Included
- `pharma_industry.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| Drug Dosage (mg) | Amount of drug dosage administered |
| Systolic Blood Pressure (mmHg) | Patient systolic blood pressure measurement |
| Heart Rate (BPM) | Patient heart rate in beats per minute |
| Liver Toxicity Index (U/L) | Liver toxicity measurement level |
| Blood Glucose Level (mg/dL) | Blood glucose concentration level |
| Drug Response | Drug effectiveness response (0 = Negative, 1 = Positive) |

---

## 🛠️ Usage
This dataset is used for:
- Machine Learning Classification
- Healthcare Data Analysis
- Drug Response Prediction
- Exploratory Data Analysis (EDA)
- Statistical Analysis
