# Dataset Information

## 📊 Dataset Name
EastWest Airlines Customer Dataset

---

## 📌 Description
This dataset contains information about EastWest Airlines customers, including their mileage balances, bonus miles, flight activity, enrollment history, and award status. The dataset is commonly used for Customer Segmentation and Clustering analysis.

---

## 📂 File Included
- `EastWestAirlines.xlsx`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| ID# | Unique customer identifier |
| Balance | Mileage balance in customer's account |
| Qual_miles | Qualified miles earned |
| cc1_miles | Credit card miles category 1 |
| cc2_miles | Credit card miles category 2 |
| cc3_miles | Credit card miles category 3 |
| Bonus_miles | Bonus miles earned |
| Bonus_trans | Number of bonus mile transactions |
| Flight_miles_12mo | Flight miles earned in the last 12 months |
| Flight_trans_12 | Number of flight transactions in the last 12 months |
| Days_since_enroll | Number of days since customer enrollment |
| Award? | Award status (0 = No Award, 1 = Award Received) |

---

## 🛠️ Usage
This dataset is used for:
- Clustering Analysis
- Customer Segmentation
- K-Means Clustering
- Hierarchical Clustering
- Exploratory Data Analysis (EDA)
- Pattern Discovery
