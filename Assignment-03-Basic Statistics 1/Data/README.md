# Dataset Information

## 📊 Dataset Name
Retail Sales Dataset

---

## 📌 Description
This dataset contains retail sales transaction details including product information, sales values, discounts, and net sales across different business units and cities. The dataset is used for Data Analysis, Statistical Analysis, and Machine Learning practice.

---

## 📂 File Included
- `dataset.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| Date | Date of transaction |
| Day | Day of the week |
| SKU | Product unique identifier |
| City | City where sale occurred |
| Volume | Quantity sold |
| BU | Business Unit |
| Brand | Product brand |
| Model | Product model |
| Avg Price | Average selling price |
| Total Sales Value | Total sales amount before discount |
| Discount Rate (%) | Discount percentage applied |
| Discount Amount | Total discount amount |
| Net Sales Value | Final sales amount after discount |

---

## 🛠️ Usage
This dataset is used for:
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization
- Machine Learning Practice
