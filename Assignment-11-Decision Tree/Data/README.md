# Dataset Information

## 📊 Dataset Name
Heart Disease Dataset

---

## 📌 Description
This dataset contains medical and patient health-related information used to predict the presence of heart disease. The dataset includes clinical parameters such as age, cholesterol levels, blood pressure, chest pain type, and other diagnostic measurements used for Machine Learning classification tasks.

---

## 📂 File Included
- `heart_disease.xlsx`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| age | Age of the patient |
| sex | Gender of the patient |
| cp | Chest pain type |
| trestbps | Resting blood pressure |
| chol | Serum cholesterol level |
| fbs | Fasting blood sugar |
| restecg | Resting electrocardiographic results |
| thalch | Maximum heart rate achieved |
| exang | Exercise-induced angina |
| oldpeak | ST depression induced by exercise |
| slope | Slope of the peak exercise ST segment |
| thal | Thalassemia condition |
| num | Heart disease diagnosis result |

---

This dataset is used for:
- Decision Tree Classification
- Machine Learning Model Training
- Healthcare Prediction

- Statistical Analysis
