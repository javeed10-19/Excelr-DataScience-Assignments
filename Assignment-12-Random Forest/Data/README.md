# Dataset Information

## 📊 Dataset Name
Glass Dataset

---

## 📌 Description
This dataset contains the chemical composition details of different types of glass. The dataset is commonly used for Machine Learning classification tasks to identify glass types based on their chemical properties.

---

## 📂 File Included
- `glass.xlsx`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| RI | Refractive Index |
| Na | Sodium content |
| Mg | Magnesium content |
| Al | Aluminum content |
| Si | Silicon content |
| K | Potassium content |
| Ca | Calcium content |
| Ba | Barium content |
| Fe | Iron content |
| Type | Type of glass classification |

---

## 🛠️ Usage
This dataset is used for:
- Random Forest Classification
- Machine Learning Model Training
- Classification Problems
- Exploratory Data Analysis (EDA)
- Statistical Analysis
