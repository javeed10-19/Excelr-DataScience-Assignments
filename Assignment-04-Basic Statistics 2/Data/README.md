# Dataset Information

## 📊 Dataset Name
Hospital Patients Dataset

---

## 📌 Description
This dataset contains hospital patient records including department details, doctor information, billing amounts, receptionist IDs, and patient check-in times. The dataset is used for practicing data analysis, data cleaning, and statistical operations in Python.

---

## 📂 File Included
- `patients.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| PatientID | Unique identifier for each patient |
| Name | Name of the patient |
| Department | Hospital department assigned |
| Doctor | Doctor handling the patient |
| BillAmount | Total hospital bill amount |
| ReceptionistID | Receptionist handling the check-in |
| CheckInTime | Patient check-in date and time |

---

## 🛠️ Usage
This dataset is used for:
- Data Cleaning
- Handling Missing Values
- Statistical Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization Practice

Dataset provided as part of the ExcelR Data Science course assignments.
