# Dataset Information

## 📊 Dataset Name
Anime Recommendation Dataset

---

## 📌 Description
This dataset contains information about various anime titles, including genres, ratings, episode counts, and community engagement metrics. The dataset is commonly used for building Recommendation Systems and analyzing user preferences.

---

## 📂 File Included
- `anime.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| anime_id | Unique identifier for each anime |
| name | Name of the anime |
| genre | Genre(s) of the anime |
| type | Type of anime (TV, Movie, OVA, Special, etc.) |
| episodes | Total number of episodes |
| rating | Average user rating |
| members | Number of community members/users who interacted with the anime |

---

## 🛠️ Usage
This dataset is used for:
- Recommendation Systems
- Content-Based Filtering
- Similarity Analysis
- User Preference Analysis
- Exploratory Data Analysis (EDA)
- Machine Learning Applications
