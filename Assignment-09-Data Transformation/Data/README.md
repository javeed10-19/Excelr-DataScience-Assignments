# Dataset Information

## 📊 Dataset Name
Adult Income Dataset

---

## 📌 Description
This dataset contains demographic and employment-related information used to predict whether an individual's income exceeds \$50K per year. The dataset is widely used for Machine Learning classification and data analysis tasks.

---

## 📂 File Included
- `adult_with_headers (1).csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| age | Age of the individual |
| workclass | Type of employment |
| fnlwgt | Final weight assigned by census |
| education | Highest education qualification |
| education_num | Numerical representation of education |
| marital_status | Marital status of the individual |
| occupation | Occupation type |
| relationship | Relationship status |
| race | Race category |
| sex | Gender of the individual |
| capital_gain | Capital gain amount |
| capital_loss | Capital loss amount |
| hours_per_week | Number of working hours per week |
| native_country | Country of origin |
| income | Income category (<=50K or >50K) |

---

## 🛠️ Usage
This dataset is used for:
- Logistic Regression
- Classification Problems
- Data Analysis
- Exploratory Data Analysis (EDA)
- Machine Learning Model Training
