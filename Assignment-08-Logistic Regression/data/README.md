# Dataset Information

## 📊 Dataset Name
Diabetes Prediction Dataset

---

## 📌 Description
This dataset contains medical diagnostic measurements used to predict whether a patient is likely to have diabetes. The dataset is commonly used for Machine Learning classification problems and healthcare prediction analysis.

---

## 📂 File Included
- `diabetes.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| Pregnancies | Number of times pregnant |
| Glucose | Glucose concentration level |
| BloodPressure | Blood pressure measurement |
| SkinThickness | Thickness of skin fold |
| Insulin | Insulin level in blood |
| BMI | Body Mass Index |
| DiabetesPedigreeFunction | Diabetes hereditary function score |
| Age | Age of the patient |
| Outcome | Diabetes prediction result (0 = No Diabetes, 1 = Diabetes) |

---

## 🛠️ Usage
This dataset is used for:
- Logistic Regression
- Classification Problems
- Machine Learning Model Training
- Data Analysis
- Healthcare Prediction
