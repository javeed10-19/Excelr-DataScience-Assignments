# Diabetes Prediction System

## 📌 Objective
This project focuses on building and deploying a Logistic Regression Machine Learning model to predict diabetes risk based on patient health parameters.

---

## 📚 Features
- Diabetes Risk Prediction
- Interactive Streamlit Web App
- Machine Learning Model Deployment
- User-Friendly Interface

---

## 🛠️ Tools & Technologies Used
- Python
- Jupyter Notebook
- Streamlit
- Pandas
- NumPy
- Scikit-learn

---

## 📂 Files Included

- `app.py`
- `diabetes_model.pkl`
- `diabetes.csv`
- `Logistic_Regression.ipynb`
- `requirements.txt`

---

## ▶️ How to Run

### Step 1: Open Anaconda Prompt

Open **Anaconda Prompt** on your system.

---

### Step 2: Navigate to Project Folder

Use the `cd` command to move to the project directory.

Example:

```bash
cd Desktop/Assignment-10-Logistic-Regression
```

OR

```bash
cd "C:\Users\YourName\Desktop\Assignment-10-Logistic-Regression"
```

---

### Step 3: Install Required Libraries

Run the following command:

```bash
pip install -r requirements.txt
```

---

### Step 4: Run the Streamlit Application

Execute the following command:

```bash
streamlit run app.py
```

---

### Step 5: Open in Browser

After running the command, Streamlit automatically opens the application in your browser.
