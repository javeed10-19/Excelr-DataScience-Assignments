# Streamlit Application

## 📌 Description
This file contains the Streamlit web application used to deploy the Logistic Regression Machine Learning model for diabetes prediction.

The application allows users to enter medical parameters and predict the risk of diabetes interactively through a user-friendly interface.

---

## 📂 File Included
- `app.py`

---

## 🛠️ Technologies Used
- Python
- Streamlit
- NumPy
- Pickle
- Scikit-learn

---

## ⚙️ Features
- Interactive User Interface
- Real-Time Diabetes Prediction
- Probability Score Display
- Logistic Regression Model Integration
- User-Friendly Input Controls

---

## ▶️ How to Run

### Step 1: Open Anaconda Prompt

Navigate to the application folder:

```bash
cd Desktop/Assignment-10-Logistic-Regression/app
```

OR

```bash
cd "C:\Users\YourName\Desktop\Assignment-10-Logistic-Regression\app"
```

---

### Step 2: Run Streamlit Application

Execute the following command:

```bash
streamlit run app.py
```

---

### Step 3: Open in Browser

After running the command, Streamlit automatically opens the web application in your browser.
