# Dataset Information

## 📊 Dataset Name
Wine Classification Dataset

---

## 📌 Description
This dataset contains the chemical composition of different wine samples. The dataset is commonly used for Machine Learning classification tasks to identify wine types based on their physicochemical properties.

---

## 📂 File Included
- `wine.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|------------|-------------|
| Type | Wine class/category |
| Alcohol | Alcohol content |
| Malic | Malic acid concentration |
| Ash | Ash content |
| Alcalinity | Alkalinity of ash |
| Magnesium | Magnesium concentration |
| Phenols | Total phenolic compounds |
| Flavanoids | Flavanoid content |
| Nonflavanoids | Non-flavanoid phenols |
| Proanthocyanins | Proanthocyanin concentration |
| Color | Color intensity |
| Hue | Hue of the wine |
| Dilution | OD280/OD315 dilution ratio |
| Proline | Proline amino acid concentration |

---

## 🛠️ Usage
This dataset is used for:
- XGBoost Classification
- LightGBM Classification
- Machine Learning Model Training
- Feature Analysis
- Exploratory Data Analysis (EDA)
- Multiclass Classification Problems
- Principal Component Analysis(PCA)
