# Dataset Information

## 📊 Dataset Name
Toyota Car Price Dataset

---

## 📌 Description
This dataset contains information about Toyota cars, including price, age, mileage, fuel type, engine specifications, and other vehicle features. The dataset is used for Multiple Linear Regression analysis to predict car prices based on multiple independent variables.

---

## 📂 File Included
- `toyota_corolla.csv`

---

## 📚 Columns / Features

| Column Name | Description |
|---|---|
| Price | Price of the car |
| Age_08_04 | Age of the car in months |
| KM | Kilometers driven |
| Fuel_Type | Type of fuel used |
| HP | Horsepower of the car |
| Automatic | Transmission type (0 = Manual, 1 = Automatic) |
| cc | Engine capacity in cubic centimeters |
| Doors | Number of doors |
| Cylinders | Number of engine cylinders |
| Gears | Number of gears |
| Weight | Weight of the car |

---

## 🛠️ Usage
This dataset is used for:
- Multiple Linear Regression
- Predictive Modeling
- Feature Analysis
- Correlation Analysis
- Machine Learning Practice
